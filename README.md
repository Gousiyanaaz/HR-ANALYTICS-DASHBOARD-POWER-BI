# HR-ANALYTICS-DASHBOARD-POWER-BI
HR Analytics Dashboard focuses on understanding employee attrition patterns and workforce trends using data-driven analysis. The project transforms HR data into clear insights that help identify key factors influencing employee turnover.
The dashboard highlights how attrition varies across different demographic and professional attributes, making it easier to recognize trends related to experience levels, compensation ranges, and job roles. By analyzing tenure and age-related patterns, the dashboard helps organizations understand at which stages employees are more likely to leave.
Overall, this project supports HR teams in making informed decisions related to retention strategies, workforce planning, and organizational improvement through interactive and easy-to-understand visual insights.
